# Apex Billing System

A modern **Smart Supermarket Billing System** built using the **MERN Stack**, where a **mobile browser acts as a real-time barcode scanner** and a **desktop browser serves as the billing dashboard**. The application enables fast billing, inventory management, customer insights, and real-time synchronization without requiring a mobile app.

## Key Features

- Real-time barcode scanning using a mobile browser
- QR code-based device pairing
- Live synchronization with Socket.IO
- Fast billing with automatic GST and discount calculation
- Customer purchase history and loyalty management
- Personalized offers and product recommendations
- Inventory management with low-stock alerts
- Sales analytics and reporting dashboard
- Secure JWT authentication with role-based access
- Printable and digital receipts

## Tech Stack

**Frontend:** React.js, Tailwind CSS, React Router, Socket.IO Client, ZXing

**Backend:** Node.js, Express.js, Socket.IO, JWT, bcrypt

**Database:** MongoDB, Mongoose

## Installation

```bash
git clone https://github.com/your-username/apex-billing.git
cd apex-billing

# Install frontend
cd client
npm install

# Install backend
cd ../server
npm install

# Start frontend and backend
npm run dev
```
